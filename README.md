# 📊 GDP & Economic Productivity Analysis Dashboard (India)

## 🚀 Project Overview
This project focuses on analyzing **India’s GDP and economic productivity** using interactive dashboards and visualizations. It combines **data analysis, Power BI dashboards, and a Python-based web app** to provide meaningful insights into economic indicators.

---

## 🎯 Objectives
- Analyze GDP trends across Indian states  
- Visualize economic productivity patterns  
- Build interactive dashboards for better decision-making  
- Provide insights using data-driven approaches  

---

## 🛠️ Tech Stack
- **Languages:** Python, SQL  
- **Tools:** Power BI, VS Code, Jupyter Notebook, Postman  
- **Libraries:** pandas, NumPy  
- **Framework:** Streamlit  

---

## 📁 Project Structure
```
├── app.py                          # Streamlit web application
├── GDP_Productivity_PowerBI.pbix   # Power BI dashboard file
├── Economy_Productivity_SD_India   # Dataset file
├── GDP_of_Indian_states.png        # Visualization output
├── economic_indicator.jpg          # Supporting image
├── seale.pbix                      # Additional Power BI dashboard
├── Images and working.zip          # Project resources
├── README.md                       # Project documentation
```
